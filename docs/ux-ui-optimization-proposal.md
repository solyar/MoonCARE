| 主色 | Primary | `#EC4899` | `pink-500` |
| 主色-亮 | Primary Light | `#F472B6` | `pink-400` |
| 主色-深 | Primary Dark | `#DB2777` | `pink-600` |
| 辅助色 | Secondary | `#8B5CF6` | `purple-500` |
| 强调色 | Accent | `#F97316` | `orange-500` |
| 背景色 | Background | `#F9FAFB` | `gray-50` |
| 卡片色 | Surface | `#FFFFFF` | `white` |

## 2. 页面结构规范

### 2.1 页面容器

所有页面统一使用以下结构：

```html
<div class="page-name">
  <div class="max-w-lg mx-auto pb-16 px-4">
    <!-- 页面内容 -->
  </div>
  <BottomNav />
</div>
```

### 2.2 页面层级

| 页面 | 容器类名 | 内边距 |
|------|---------|--------|
| 首页 | `home-page` | `px-4 pt-4 pb-3` |
| 日记 | `diary-page` | `px-4 pt-4 pb-3` |
| 周期 | `cycle-page` | `px-4 pt-4 pb-3` |
| 我的 | `profile-page` | `px-4 pt-4 pb-3` |

## 3. 组件规范

### 3.1 卡片组件

```html
<div class="bg-white rounded-xl shadow-sm border border-gray-100 p-4">
  <!-- 卡片内容 -->
</div>
```

### 3.2 按钮组件

**主按钮:**
```html
<button class="px-4 py-2 bg-pink-500 text-white rounded-full text-sm font-medium">
  按钮文字
</button>
```

**次要按钮:**
```html
<button class="px-4 py-2 bg-gray-100 text-gray-600 rounded-full text-sm font-medium">
  按钮文字
</button>
```

### 3.3 底部导航

```html
<nav class="fixed bottom-0 left-0 right-0 bg-white border-t border-gray-200 z-50">
  <div class="flex justify-around items-center h-14 max-w-lg mx-auto">
    <!-- 导航项 -->
  </div>
</nav>
```

**导航项状态:**
- 默认: `text-gray-400`
- 选中: `text-pink-500` + 渐变色顶部指示条

## 8. 动效规范

### 8.1 过渡时长

| 动效类型 | 时长 | Class |
|---------|------|-------|
| 快速 | 150ms | `transition-none` |
| 正常 | 200ms | `transition-all duration-200` |
| 慢速 | 300ms | `transition-all duration-300` |
| 进度条 | 500ms | `transition-all duration-500` |

### 8.2 触摸反馈

| 元素 | 动效 |
|------|------|
| 卡片点击 | `active:scale-[0.98]` |
| 按钮点击 | `active:scale-95` |
| 底部导航图标 | 选中时 `scale-110` |
